Princípios metodológicos de programação

A formação em Princípios metodológicos de programação permitem o desenvolvimento de competências tendo em vista o reconhecimento das vantagens e desvantagens da utilização de uma metodologia.

As principais metodologias no desenvolvimento de software são:

  1. Metodologia Ágil
  2. Metodologia Cachoeira
  3. Metodologia DevOps
  4. Metodologia Aplicação rápida
  5. Metodologia Espiral.
  6. Metodologia Protótipo.
  7. Metodologia Programação extrema (XP)
  8. Metodologia Desenvolvimento orientado a recursos.
  9. Metodologia Desenvolvimento de aplicativos conjuntos
